Twitter Bootstrap

The fastest way to prototype

Twitter Bootstrap is a powerful front-end framework for faster web development(this reference guide was created using Twitter Bootstrap). It includes HTML and CSS based design templates for common user interface components like fonts, forms, buttons and many other web controls. Twitter Bootstrap also gives you the ability to create responsive layouts without having to know too much about grid-based design principles.

Advantages in using Twitter Bootstrap

The biggest advantage of using Twitter Bootstrap is that it comes with a free set of tools for creating flexible and responsive web layouts as well as common interface components. On top of providing these components, it also offers a lot of existing JavaScript functionality that can be easily integrated into your web pages. Below are just a few of the advantages in using Twitter Bootstrap over another front-end framework:

  • Save Time — You can save considerable amounts of time and effort using the Bootstrap pre-defined templates, CSS classes, etc.
  • Clear & Consistent Design — All Bootstrap components share the same design templates and styles through a central library. This ensures that the designs and layouts of your web pages are consistent throughout an entire project/application.
  • Ease of Use — Twitter Bootstrap is very easy to use. Having a basic knowledge of HTML and CSS can get you started very quickly. The components page has great documentation which specifies how web controls should be used, styled, etc.
  • Browser Compatibility — Twitter Bootstrap is created with modern browsers in mind and is compatible with all modern browsers(Google Chrome, Mozilla Firefox, Safari, & Internet Explorer).
  • Open Source — The framework is completely free of charge. You can take it and integrate it into your project without having to deal with additional licensing costs. Also, as it's very popular within the open source community, developers are continually developing plug-ins that will help increase the usage of Twitter Bootstrap.
