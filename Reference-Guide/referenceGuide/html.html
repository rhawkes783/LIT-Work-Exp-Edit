HTML - Hypertext Markup Language

The backbone of the internet

HTML stands for HyperText Markup Language. Developed by scientist Tim Berners-Lee in 1990, HTML is the code that helps us communicate with others on the world wide web.

When writing HTML, you add "tags" to the text in order to create the structure. These tags tell the browser how to display the text or graphics in the document. All tags should be opened and closed correctly. For example, the following document has a simple layout. Notice there are three major parts: a heading, two paragraphs and a bulleted list. 



<html>
    <h1>This is heading text</h1>
    <p>This is a section of paragraph text</p>
    <p>This is another section of paragraph text :)</p>
    <!-- This is an unordered list element. It contains list items and displays them onto the page as bullet points by default -->
    <ul>
        <li>first listitem</li>
        <li>second listitem</li>
        <li>third listitem</li>
    </ul>
    <!-- End of unordered list -->
</html>

The code above will create the following output when viewed in a web browser.

The tools you need

Fortunately, HTML is written in plain text. That means you aren't required to spend lots of money on editing software to create HTML. All you need to create a document is a simple text editor. Every operating system comes with a text editor pre-installed(If you are using Windows, it's Notepad. If you are using an Apple computer, it's TextEdit). One of the most popular free text editors available right now is Sublime Text, which is available for Windows and Apple machines.

The basics

As with everything code-related, there are rules which we as developers must abide by. In HTML, the rules are fairly simple. For starters, HTML tags are always surrounded by what are called angle brackets < and >.

For example: 

                
                
<p>This is how to construct a paragraph in HTML</p>

It's important to note that if you forget to include tags when writing HTML, the web page won't appear as you would like it to. Fortunately, with most text editors(especially Sublime Text), it's easy to spot where an angle bracket is missing, etc.

Elements

The words and letters between these two angle brackets are called elements. These are the coded commands within HTML, which tell the browser how to display the web page. Below is a basic example of a header element followed by a span element. 

                    
<h1>I am a header element</h1>
<span>I am a span element</span>

Examples of elements

<a>

The <a> element is known as an HTML anchor. It creates a hyperlink.


<a href="http://www.google.com/">
Link to Google
</a>

<body>

The <body> element represents the content of the HTML document. Unlike other HTML elements, the <body> element can only appear once in a document.


<body>
<!-- All content goes here -->
</body>

<button>

The <button> element creates a button on the document.


<button id="btn">Click here</button>

<hr>

The <hr> element creates a horizontal rule in the document. It is normally used to create a line between paragraphs on an HTML page.


<p>This is the first paragraph.</p>
<!-- horizontal rule used to break up space between paragraphs and add line to document -->
<hr>
<p>This is the second paragraph.</p>

This section simply explains a few of the most basic elements used in HTML and how they can be used. If you want to know more about all of the elements currently available in HTML, the Mozilla Developer Network is a good place to read up on things. MDN is maintained by the developers who created Mozilla FireFox and their learning materials are incredibly beneficial for web developers.

Page Structure

HTML essentially provides the page with content and structure. What it doesn't provide is styling; that's covered by Cascading Style Sheets. Every page on the web will boil down to pure HTML, no matter how fancy it may appear in your web browser. If you'd like to view the page structure of a web page currently online, simply follow the steps below.

  • Open up your web browser
  • Navigate to a website(eg. http://marvel.com/guardians)
  • Once the page has loaded, right-click and select 'View page source'

This will give you access to the HTML of your favourite websites and let you have a closer look at what professional web designers and developers are doing with HTML.
